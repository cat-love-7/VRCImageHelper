# VRCImageHelper

[m-hayabusa/VRChat-Exif-Writer: add EXIF tag to VRChat Pics: https://github.com/m-hayabusa/VRChat-Exif-Writer](https://github.com/m-hayabusa/VRChat-Exif-Writer) をC#で書きなおしています。

* インストーラをビルドしたあと、`&"C:\Program Files (x86)\Windows Kits\10\bin\10.0.22000.0\x86\MsiInfo.exe" .\Setup.msi -w 10` とすると管理者権限を必要とせずにセットアップできる
